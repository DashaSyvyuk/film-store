Архітектура прикладення:
/config - у даній директорії знаходяться файли з налаштуваннями різних параметрів сайтів, а також дані для входу в базу даних
	
/controllers - у цій директорії знаходяться контролери для всіх сторінок, для переходу на необхідну сторінку необхідно до адреси сайту дописати назву контролера

/libs - тут знаходяться основні три класи Model, View та Controller, а також всі класи які не входять у формування моделі MVC
	Bootstrap.php - клас, який визначає як саме будуть організовані переходи за різними посилання і які дані є назвою контролера,
 			методу контролера, а які будуть параметрами метода
	Database.php  - клас для підключення до Бази Даних
	Pagination.php - клас, що повертає html код пагінації сторінки

/models - у цій директорії знаходяться відповідні моделі для кожного контролера, якщо необхідна робота з БД. Назва моделі формується
	  з таких частин назваконтролера_model.php
/public - тут знаходяться загальні css та js файли, код яких підключається на всіх сторінках сайту. Також є папка downloads, 
	   у яку завантажуються файли з інформацією про фільм
/views  - у цій директорії знаходиться html код сторінок. Код, який стосується лише конкретного контролера знаходиться у відповідній папці,
	  де є ще файли css та js, які використовуються лише на сторінці конкретного контролера.
	  Якщо розмітка використовується на кількох сторінках, то файл із нею винесений у корінь.
.htaccess - знаходяться налаштування сервера
index.php - основний файл, який запускається перший. У ньому підключаються всі необхідні файли і створюється клас Bootstrap

Запуск прикладення
1. У файлі config/database.php змінити дані для підключення до Бази даних
2. Командою php -S localhost:8000 запустити вбудований сервер php
3. Запустити Базу Даних
4. Виконати запит до бази даних із файлу create_table.txt
5. У адресній строці браузера ввести http://localhost:8000 і почати користуватися прикладенням
